# 2-bit ALU using Verilog

## Objective

Design a 2-bit Arithmetic Logic Unit (ALU) to perform basic arithmetic and logical operations.

## Tools Used

* Verilog HDL
* ModelSim

## Operations Performed

* Addition
* Subtraction
* AND
* OR
* XOR

## Files Included

* alu.v → ALU design
* alu_tb.v → Testbench
* run.do → Simulation script
* waveform.png → Output waveform

## Output

Simulation waveforms verify correct operation of ALU.
